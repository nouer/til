# Mistel Barocco MD650L
## 構成
- Cheery ML Low Profile

## 設定
- QWERTY, Clemak, Dvorkの切り替え
  - Ctrl+' (MD600) は Ctrl+A
- レイアウト切り替え
  - Fn + m = レイヤー 0
  - Fn + < = レイヤー 1
  - Fn + > = レイヤー 2
  - Fn + / = レイヤー 3
- 初期化
  - 初期のFn + r を5秒押しっぱなし
- キーの変更
  1. 変更開始  
    Fn + RightCtrl
  1. 変更されるキーを打鍵
  1. 指定するキーを打鍵  
  1. pnキーを打鍵  
  1. 変更終了  
    Fn + RightCtrol
- Dip Switch
  - Switch 1 = Caps <-> Fn置換
  - Switch 2 = Caps <-> RightCtrl置換
  - Switch 3 = LeftMeta <-> LeftAlt置換
  - Switch 4 = LeftSpace <- Fn置換（Fnが２つになる）
